Glosario de Etiquetas HTML

<html>
Raíz del documento HTML. Todos los demás elementos deben ser descendientes de este elemento.
<head>
Cabecera del documento HTML. Contiene metadatos sobre el documento.
<body>
Cuerpo del documento HTML. Contiene todo el contenido visible de la página.
<h1> a <h6>
Encabezados de diferentes niveles, siendo h1 el más importante y h6 el menos importante.
<p>
Párrafo de texto.
<a>
Enlace a otra página web o a una sección dentro de la misma página.
<img>
Imagen que se muestra en la página web.
<ul>
Lista no ordenada. Se utiliza para crear listas con viñetas.
<ol>
Lista ordenada. Se utiliza para crear listas numeradas.
<div>
Contenedor genérico para agrupar contenido. Se utiliza frecuentemente para aplicar estilos CSS.
